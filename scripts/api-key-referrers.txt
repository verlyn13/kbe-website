# HTTP Referrers to Add for KBE Website API Key

Copy and paste each of these lines into the "Website restrictions" field in Google Cloud Console:

https://homerconnect.com/*
https://www.homerconnect.com/*
https://kbe-website.firebaseapp.com/*
https://kbe-website.web.app/*
https://kbe-website--kbe-website.us-central1.hosted.app/*
http://localhost:9002/*
http://localhost:3000/*
http://localhost/*

# Important Notes:
- Add each URL on a separate line
- Include the /* at the end of each URL
- Don't add query parameters or fragments
- The wildcard /* allows all paths under that domain
